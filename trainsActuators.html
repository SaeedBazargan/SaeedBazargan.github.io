Highschool engineering projects:

  1. Mecnaum Drivetrain Assembly

    2022 - 2023

    This robot is equipped with a Mecanum wheel drivetrain, offering omnidirectional maneuverability. Integrated within its system is an advanced ultrasonic sensor array, designed for dynamic obstacle avoidance to prevent collisions with objects and walls. Further augmenting its precision navigation capabilities, the robot employs a sophisticated PID (Proportional-Integral-Derivative) control algorithm, ensuring optimal path accuracy and straight-line traversal.

  2. Multi-Level Block Retrieval and Delivery Robot

    2021 - 2022

    This robot is equipped with a Mecanum wheel drivetrain, offering omnidirectional maneuverability. Integrated into its design is a precision-engineered linear actuator mounted on an advanced elevator mechanism. This configuration facilitates vertical articulation, enabling the robot to efficiently reach and operate at multiple elevation tiers for the targeted placement and retrieval of designated blocks.

  3. Dual motor Chassis with built-in motor encoders

    2021 - 2022

    Robotpy (python) was used to operate this robot using a playstation 4 controller.

  4. Scissors Mechanism

    2019 - 2020

    This scissors mechanism was made with custom 3D-printed connectors affixed to the termini of aluminum extrusions. These extrusions are centrally connected with a bearing,enabling the mechanism to achieve a compact form factor in its fully retracted state.

  5. Linear Actuator

    2019-2020

    Simple linear actuator developed by using a very simple gearbox to increase torque for lifting the robot.

  6. Intelligent fan Arduino project

    2019-2020

    This project detects the temperature in a room, prints it on LCD screen, then operates the servo (acting as a fan) when the temperature is above the threshold provided.

  7. Servo Block Grabber

    2019 - 2020

    This is a Lego block grabber controlled by a servo.

  8. Potenciometer LED Project

    2019 - 2020

    Simple project turning on LEDS based on input from potenciometer.
